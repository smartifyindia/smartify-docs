## Technical Overview

A Wi-Fi IR blaster with integrated temperature and humidity sensors and an analog-style display. Transmits IR commands to legacy devices while simultaneously reporting real-time environmental data for automation logic.

Connects to your 2.4 GHz Wi-Fi network. Sensor data can be used as automation triggers — for example, activating an air conditioner when temperature exceeds a threshold.

Key Specifications
- Protocol: Wi-Fi 2.4 GHz

- Sensors: Temperature + Humidity
- Display: Analog-style (temp/humidity)
- Installation: Indoor, line-of-sight placement

⚠ IR line-of-sight to target devices is required. Does not switch electrical loads.
