## Overview

If something moves, you'll know.

A knock on the safe. A window forced open. A cabinet disturbed. The Vibration Sensor catches what cameras miss — and tells you instantly, before anything else happens.

## Technical Overview

A Zigbee 3.0 vibration sensor that detects movement, vibration, and tampering on doors, windows, safes, cabinets, and other surfaces, reporting events to a Zigbee gateway or compatible hub.

Used for security monitoring, tamper alerts, and automation triggers. Battery-powered with surface mount installation.

Key Specifications
- Protocol: Zigbee 3.0
- Detection: Vibration and motion
- Power: Battery
- Mounting: Surface mount (adhesive or screws)

⚠ Event notification only — does not control electrical loads.
