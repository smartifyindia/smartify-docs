## Technical Overview

A dedicated gateway that integrates VRV/VRF HVAC systems into a Zigbee-based smart automation ecosystem. Connects to the VRV/VRF outdoor unit via LAN cable and communicates wirelessly with a Zigbee gateway.

Enables remote control, real-time monitoring, and automation of the entire VRV/VRF air conditioning system through the Zigbee automation platform — without modifying the HVAC hardware.

Key Specifications
- Device Type: VRV/VRF to Zigbee Integration Gateway
- HVAC Connection: Wired LAN to outdoor unit
- Wireless: Zigbee 3.0 (to Zigbee gateway)
- Installation: Indoor, wired to outdoor unit

⚠ Communication bridge only — does not switch electrical loads. VRV/VRF system compatibility must be confirmed before installation.
